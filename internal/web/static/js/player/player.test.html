How to test reconnect (the whole point of this page)
  1. Paste any working ICY/MP3 stream URL above & hit Play. State should go to playing.
  2. Kill the stream upstream (e.g. ./grimnir down, or block the host with sudo iptables -A OUTPUT -d <ip> -j DROP). You should see one or two reconnect log lines, then state stays reconnecting.
  3. Bring the stream back. State should return to playing within a second or two.
  4. Kill the stream and leave it dead. The reconnect counter should hit 3, the Escalations counter should increment, & the log should print the escalate payload. (Chunk 3 wires this to multi-URL degrade; here it just logs.)
  5. Try the Pause button mid-playback & mid-reconnect. Pause should freeze the player & stop the reconnect loop.
